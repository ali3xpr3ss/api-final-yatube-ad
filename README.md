# Yatube API - REST API для социальной сети

RESTful API для платформы социальной сети на базе Django и Django REST Framework.

## 📋 Описание проекта

**Yatube API** предоставляет полнофункциональный REST API для управления постами, комментариями, группами и подписками пользователей. API использует JWT-аутентификацию и защищены правами доступа.

## 🛠️ Технологический стек

- **Django** 3.2.16 — веб-фреймворк
- **Django REST Framework** 3.12.4 — фреймворк для REST API
- **Django REST Framework Simple JWT** 4.7.2 — JWT-аутентификация
- **SQLite** — база данных
- **pytest** 6.2.4 — тестирование
- **Pillow** 9.3.0 — обработка изображений

## 📚 Модели данных

### Post (Пост)
- `text` — содержание поста
- `pub_date` — дата публикации (автоматически)
- `author` — автор поста (ForeignKey на User)
- `image` — изображение (опционально)
- `group` — группа (опционально)

### Comment (Комментарий)
- `text` — текст комментария
- `author` — автор комментария
- `post` — связанный пост
- `created` — дата создания

### Group (Группа)
- `title` — название группы
- `slug` — уникальный идентификатор
- `description` — описание

### Follow (Подписка)
- `user` — пользователь, который подписывается
- `following` — пользователь, на которого подписываются
- Ограничение: уникальная пара (user, following)

## 🔌 API Эндпоинты

### Посты
- `GET /api/v1/posts/` — список всех постов (с пагинацией)
- `POST /api/v1/posts/` — создать пост
- `GET /api/v1/posts/{id}/` — получить пост
- `PUT/PATCH /api/v1/posts/{id}/` — редактировать пост
- `DELETE /api/v1/posts/{id}/` — удалить пост

### Комментарии
- `GET /api/v1/posts/{post_id}/comments/` — список комментариев поста
- `POST /api/v1/posts/{post_id}/comments/` — добавить комментарий
- `GET/PUT/PATCH/DELETE /api/v1/posts/{post_id}/comments/{id}/` — операции с комментарием

### Группы
- `GET /api/v1/groups/` — список групп (только чтение)
- `GET /api/v1/groups/{id}/` — получить группу

### Подписки
- `GET /api/v1/follow/` — мои подписки
- `POST /api/v1/follow/` — подписаться на пользователя
- `GET /api/v1/follow/?search=username` — поиск подписок

### JWT Аутентификация
- `POST /api/v1/jwt/create/` — получить токены
- `POST /api/v1/jwt/refresh/` — обновить access токен
- `POST /api/v1/jwt/verify/` — проверить токен

## 🔐 Права доступа

- **Посты**: создание только для авторизованных; редактирование только автором; чтение для всех
- **Комментарии**: создание только для авторизованных; редактирование только автором; чтение для всех
- **Группы**: чтение для всех (только чтение)
- **Подписки**: доступ только для авторизованных пользователей
- **JWT**: требуется токен для защищённых эндпоинтов

## 🚀 Установка и запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Применение миграций
```bash
cd yatube_api
python manage.py migrate
```

### 3. Создание суперпользователя (опционально)
```bash
python manage.py createsuperuser
```

### 4. Запуск сервера
```bash
python manage.py runserver
```

Сервер будет доступен по адресу: `http://127.0.0.1:8000/`

API доступен по адресу: `http://127.0.0.1:8000/api/v1/`

## 🧪 Тестирование

Запуск всех тестов:
```bash
pytest -vv
```

Запуск конкретного теста:
```bash
pytest tests/test_post.py -vv
```
